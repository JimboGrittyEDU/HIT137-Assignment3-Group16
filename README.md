# HIT137 Assignment 3 - Tkinter + OOP + OpenCV Image Editor

## How to run
1. Install the Python 3.14
2. Install dependencies
   - pip install -r requirements.txt
3. Run:
   - python main.py

## Features of the program
- Tkinter GUI: menu bar (File, Edit, Rotate, Flip), canvas image display, control panel, status bar
- OOP ImageModel (state/history), ImageProcessor (OpenCV ops), ImageEditorApp (GUI/controller)
- OpenCV filters:
  - Grayscale
  - Blur (slider)
  - Edge detection (slider)
  - Brightness (slider)
  - Contrast (slider)
  - Rotate 90/180/270 degrees
  - Flip horizontal/vertical
  - Resize scale %-
- Undo/Redo
